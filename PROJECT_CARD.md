#Паспорт курсового проекта

## Тема
Разработка проектного решения профессиональной социальной сети ProNet

## Проблема
На рынке трудоустройства пользователи вынуждены использовать сразу несколько сервисов для профессионального взаимодействия и для поиска работы отдельно. Нету сервиса который выолняет функционал профессиональной социальной сети, где пользователь может и искать работу/соискателей и создавать социальные связи между друг другом

# Обьект и предмет
Обьект: профессиональная социальная сеть для взаимодейтсивя специалистов, соискателей, работодателей и HR

Предмет: проектирование структуры, архитектуры, пользовательских сценариев, требований, метрик, интерфейса, моделей данных профессиональной социальной сети ProNet

## Цель

Подготовить курсовой проект ProNet как информационную систему для профессионального нетворкинга и трудоустройства, включающую требования, диаграммы, модель данных и пользовательский прототип.

## Основные задачи 
1. Проанализировать предметную область, конкурентов, и похожие сервисы
2. Определить целевую аудиторию, основные проблемы и варианты взаимодействий пользователей
3. Разработать структуру хранения данных
4. Разработать прототип интерфейса ключевых экранов приложения

## Ожидаемый результат
Полученные модели, структуры, интерфейсы, диаграммы и прототип готовой программы
