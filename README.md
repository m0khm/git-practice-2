## Назначение проекта
ProNet — это проект профессиональной социальной сети, предназначенной для взаимодействия специалистов, соискателей, работодателей и HR.
Курсовая работа решает задачу проектирования сервиса, в котором пользователь может вести карьерный профиль, просматривать профессиональную ленту, искать вакансии, отправлять отклики и отслеживать их.
Проект объединяет функции социальной сети и сервиса для трудоустройства в одном

## Состав репозитория
- docs/ — проектная документация и методика;
- src/ — исходные файлы, прототипы или скрипты;
- data/sample/ — небольшой пример данных;
- experiments/ — журнал проверок, расчётов или экспериментов;
- assets/ — изображения, схемы, диаграммы.

## План выполнения
1. Постановка задачи и анализ предметной области.
2. Подбор источников и требований.
3. Разработка метода, модели или прототипа.
4. Проверка результата.
5. Подготовка пояснительной записки.

## Текущий статус
На текущем этапе выполнена значительная часть проектной работы. Сформулированы тема, проблема, цель, объект, предмет и задачи курсовой работы. Проведён анализ предметной области, целевой аудитории, пользовательских проблем и аналогов. Подготовлены функциональные и нефункциональные требования, бизнес-правила, пользовательские сценарии и описание ключевых процессов системы.
